# Security Policy

## Reporting a Vulnerability

### Upstream (Kubernetes)

This project is a downstream fork of
[kubernetes-sigs/cluster-api-provider-azure](https://github.com/kubernetes-sigs/cluster-api-provider-azure).

For vulnerabilities in upstream Cluster API Provider Azure, follow the
[Kubernetes Security and Disclosure Information](https://kubernetes.io/docs/reference/issues-security/security/#report-a-vulnerability)
process.

### Red Hat

For vulnerabilities specific to this fork or its Red Hat distribution,
report them through the
[Red Hat Product Security](https://access.redhat.com/security/team/contact/)
portal.

Do **not** file public GitHub issues for security vulnerabilities.
